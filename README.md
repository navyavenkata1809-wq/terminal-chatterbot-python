Terminal Chatbot using Python and ChatterBot
Setup Instructions

Create a virtual environment

python -m venv chatbot_env

Activate environment

Windows:

chatbot_env\Scripts\activate

Install dependencies

pip install -r requirements.txt

Train the chatbot

python train.py

Run chatbot

python chat.py

Example:

user: Good morning! How are you doing? bot: I am doing very well, thank you for asking.

user: You're welcome bot: Do you like hats?
